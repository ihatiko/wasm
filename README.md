# DOCX Editor - Vue.js приложение

Веб-приложение для создания и редактирования DOCX документов на базе библиотеки [docx-rs](https://github.com/bokuweb/docx-rs) (docx-wasm) и Vue.js.

## Возможности

- ✅ Добавление секций в документ
- ✅ Добавление и редактирование параграфов
- ✅ Форматирование текста (жирный, курсив, подчеркнутый)
- ✅ Настройка размера шрифта
- ✅ Сохранение документа в формате .docx на диск
- ✅ Современный и удобный интерфейс

## Установка

```bash
# Установка зависимостей
npm install
# или
pnpm install
```

## Запуск

```bash
# Запуск dev-сервера
npm run dev
# или
pnpm dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
# или
pnpm build
```

Собранные файлы будут в директории `dist/`

## Использование

1. Нажмите "Добавить секцию" для создания новой секции
2. В каждой секции можно добавлять параграфы
3. Редактируйте текст и применяйте форматирование
4. Нажмите "Сохранить документ" для скачивания .docx файла

## Технологии

- **Vue.js 3** - фреймворк для создания интерфейса
- **docx-wasm** - библиотека для создания DOCX файлов (WebAssembly версия docx-rs)
- **file-saver** - для сохранения файлов на диск
- **Vite** - сборщик и dev-сервер
